Circuit Builder

Build simple electric circuits with battery, resistor, capacitor, inductor, switch, wires, labels, and level tags.

How to use:
1. Select a component and click canvas.
2. For wire/arrow-like connector, click start point then end point.
3. Drag components to move.
4. Select component and rotate/delete/edit text.
5. Use levels for Level 1, Level 2, etc.
6. Export final circuit as PNG.
